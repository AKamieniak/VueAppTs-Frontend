# VueAppTs-Frontend

### Using:

- Vue.js with TypeScript
- Vuex state management pattern
- Vue Router
- Vue I18n
- Axios to consume API
- BootstrapVue
- @vuex-module-decorators
- @vue-property-decorator
- HTML5, SCSS

### Tools:
- Visual Studio Code
- Vue Devtools

#
### Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
